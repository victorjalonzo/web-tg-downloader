Features

Requirements

What you need to get started

Ensure you have Python 3.8+ installed on your system. The application requires the following dependencies which will be installed automatically:

  • pyaes==1.6.1
  • pyperclip==1.9.0
  • pyrotgfork==2.1.35
  • PySocks==1.7.1
  • PyTgCrypto==1.2.6
  • python-dotenv==1.0.1
  • TgCrypto==1.2.5
  • uvloop==0.19.0

Telegram API Credentials

You'll need to create a Telegram API application to get your API_ID and API_HASH.

Get API Credentials

Usage

Getting started is simple

Create .env File

Create a .env file in the application folder with your Telegram credentials:

#Replace with your Telegram application credentials
API_ID="YOUR_API_ID"
API_HASH="YOUR_API_HASH"

Run the Application

Start the bot with the following command:

python app.py
